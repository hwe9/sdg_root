Database Service
Dieser Service ist eine zentrale Schnittstelle für alle Datenoperationen.

Aufgaben:

PostgreSQL-Wrapper: Bietet eine API für die Speicherung und den Abruf von Metadaten, wie z. B. Quelleninformationen, Tags und Nutzerdaten.

Weaviate-Schnittstelle: Bietet eine API für die effiziente Vektorsuche, die für die Abfrage von semantisch ähnlichen Dokumenten unerlässlich ist.

Datenkonsistenz: Stellt die Integrität der Daten über beide Datenbanken hinweg sicher.
